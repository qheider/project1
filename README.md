# Personal Profile Flask App

A minimal Flask app that shows a personal profile page and ai powered ask page

Files:
- `app.py` - Flask application
- `templates/profile.html` - Jinja template
- `static/styles.css` - styles

Run locally:

```powershell
python -m venv .venv; .\.venv\Scripts\Activate.ps1; pip install -r requirements.txt; python app.py
```

Run tests:

```powershell
.\.venv\Scripts\Activate.ps1; pytest -q
```
